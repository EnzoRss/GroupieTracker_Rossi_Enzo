Jikan API

Unofficial MyAnimeList API

Jikan est une API qui scrap le site MyAnimeList

MyAnimeList

Site qui contient tout les infos sur les anime il permet aussi de crée des liste d'anime vue avec un système de notation pour chaque anime ajouté a sa liste.

Comment fonctionne L'API

L'API fonctionne avec du scrapping des infos du site MyAnimeList

Pourquoi Jikan et pas MyAnimeList ?

L'API de Jikan est plus simple elle ne demande pas d'autentification , elle est aussi plus complète elle possède une partie REST qui ajoute beaucoup de paramètre en plus
